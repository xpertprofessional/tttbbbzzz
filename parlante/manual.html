Manual

This manual describes all major functions, supported formats, specifications, advanced settings, menus, and keyboard shortcuts for Audio Waveform Editor. The app is a native macOS audio editor with tile-based waveform display, selection-based editing, and export to multiple formats.

Import & export formats

Legal import formats

You can open the following file types via File → Open (⌘O), drag-and-drop, or the toolbar Open button.

  • MP3 — MPEG-1/2 Layer 3 audio
  • WAV — Waveform audio (PCM)
  • AIFF — Audio Interchange File Format
  • M4A / MPEG-4 Audio — AAC and similar codecs in MP4 container
  • FLAC — Free Lossless Audio Codec
  • Movie / QuickTime — Video files; the app extracts the audio track for editing (formats supported by AVFoundation)

Multiple files can be selected at once; they are merged into a single document with optional silence between them (configurable in Advanced Settings).

Export formats

Export and Extract use the format and options set in Settings → Export. The following output formats are supported:

  • WAV — Uncompressed PCM; supports 8, 16, 24, 32 bit depth and configurable sample rate and channels
  • AIFF — Uncompressed; same bit depth and sample rate options as WAV
  • MP3 — Compressed; bitrate selectable (96–320 kbps)
  • M4A — Compressed (AAC); bitrate selectable
  • FLAC — Lossless compressed; bit depth and sample rate options

When Lossless extraction is enabled for an unedited MP3 source, the app can export by copying original MP3 frames (no re-encode). In that case only MP3 export is used and format/sample-rate/bit-depth controls are disabled.

Specifications

Export format options

  • Sample rate — Match source, or 8000, 11025, 16000, 22050, 32000, 44100, 48000, 88200, 96000 Hz
  • Bit depth (WAV, AIFF, FLAC) — Match source, or 8, 16, 24, 32 bit
  • Bitrate (MP3, M4A) — 96, 128, 160, 192, 256, 320 kbps
  • Channels — Match source, Mono, Stereo, Left only, Right only, Mid only (L+R)/2, Side only (L−R)/2

Naming & location

  • Export directory — Default is same folder as source file; or set a preferred folder in Export settings (stored as a security-scoped bookmark).
  • Filename suffix — Applied to exported/extracted names (e.g. _extract).
  • Unique names — Counter (file_1, file_2) or Timestamp to avoid overwriting.

Interface & views

Main window layout

From top to bottom: Header toolbar, Waveform display area (with optional mini waveform overview above it), Selection info bar, and optionally the Edit operations toolbar.

Header toolbar

  • Open — Opens file picker (⌘O).
  • Transport — Play/Pause (Space), Play selection only (⌥Space), Stop (⌘.). Volume control is on the right.
  • Zoom controls — Zoom in/out, reset zoom, zoom to selection (⌘\).
  • Waveform view mode (stereo only) — Switch between Left/Right channels and Mid/Side (⌘⌥L / ⌘⌥S).

Waveform display area

The main waveform is drawn with Metal (GPU), using a tile-based cache so zoom and pan stay smooth. You can:

  • Click-drag on the waveform to set a selection.
  • Scroll / trackpad to pan and zoom (behavior can be selection-centered in View options).
  • When scrubbing is enabled (⌘J), drag horizontally to scrub through audio (optional turntable-style velocity scrubbing in Playback menu).

Visual elements: Waveform (customizable color), selection overlay (color and opacity in Appearance), red playhead line, yellow split-point markers (when splits are added). In scrubbing mode, the scrubbing indicator uses the scrubbing color (default purple).

Mini waveform overview

An optional strip above the main waveform shows the full file; the visible time range is highlighted. Toggle with View → Show/Hide Mini Waveform (⌘⇧M). Height is adjustable in preferences.

Selection info bar & handles

When a selection exists, the bar below the waveform shows:

  • Start / End time fields — Monospaced; editable for precise selection. ⌘⌥1 jumps focus to the start field.
  • Clear — Clears selection (same as Escape or ⌘⇧A).
  • Split — Adds a split point at the playhead (⌘P). Split points divide the selection into segments for multi-part extract.
  • Snap to zero crossings — Optional button; or use Edit → Snap Selection to Zero Crossings (⌘⌥Z) to avoid clicks at boundaries.
  • Loop — Toggle loop playback over selection (⌘L).
  • Extract — Export selection to a new file (⌘E). Command-click or use ⌘⇧E to extract and open in a new window.

Selection handles: The selection has left and right edges. You can drag these edges to adjust the range. When Dual Edit Mode is enabled (experimental), modifier+drag on the handles adjusts varispeed (time stretch) or gain (see Edit menu and Shortcut library). Lock selection (⌘⇧L) to prevent accidental moves; when locked, split points and boundary moves are disabled until you unlock.

Playback & navigation

  • Play / Pause — Space. Plays from playhead; if “Play selection only” is used (⌥Space), playback is limited to the selection and can auto-center the view (toggle in View menu).
  • Stop — ⌘. stops playback; Enter stops and seeks to start.
  • Seek to start/end — Home / End.
  • Seek by step — Left/Right arrows move by the current movement precision (Playback → Movement Precision or ⌥1–⌥9). Option+Left/Right cycles the precision level.
  • Pan viewport — ⇧← / ⇧→. Viewport zoom: ⇧↑ zoom in, ⇧↓ zoom out (centered on viewport).
  • Playhead follow — When enabled (⌘⇧F), the view scrolls to keep the playhead visible during playback.
  • Scrubbing — ⌘J toggles scrub mode; then drag on the waveform to hear the audio at the drag position. Optional turntable mode (Playback → Enable Turntable Mode) uses drag velocity for scrub speed. Scrubbing preview duration is set in Advanced Settings.
  • Varispeed — If enabled in Advanced Settings, the varispeed control (e.g. in header or dedicated control) changes playback speed/pitch without editing the file. Range and step mode (percentage, musical semitones, or movement precision) are configurable.

Selection & handles

Creating and changing selection

  • Click-drag on the waveform to set a new selection.
  • ⌘A selects the entire file.
  • Escape or ⌘⇧A clears the selection.
  • ⌘1 / ⌘2 set selection start or end to the current playhead position.
  • [ and ] seek to selection start and end.
  • ⌘⇧← / ⌘⇧→ move the selection start by the movement precision; ⌘⌥← / ⌘⌥→ move the selection end.

Lock, splits, zero crossings

  • Lock selection (⌘⇧L) — Locks the selection and split points so they can’t be moved or deleted until unlocked. Useful before doing multiple operations.
  • Split points — ⌘P adds a split at the playhead; ⌘⇧P clears all splits. Shift+click on a split removes it. Splits divide the selection into segments (e.g. for “Extract” producing multiple files or segments). “Split by longest segment” and “Redistribute splits equally” are in the Selection menu.
  • Snap to zero crossings (⌘⌥Z) — Moves selection start and end to the nearest zero crossing to reduce clicks at edit boundaries. Optional “Snap to zero crossing” in preferences can affect other operations.

Channel selection (stereo)

When a selection is active, Playback → Channels (or ⇧1/2/0/3/4) sets which channel(s) the selection refers to: Left, Right, Both, Mid (L+R)/2, or Side (L−R)/2. This affects which channels are used for edit and export when channel configuration is “match source” or when exporting specific channel options.

Edit operations

Edit operations require “Enable Edit Operations” in Settings (Experimental). The edit toolbar can be shown/hidden with View → Toggle Edit Operations (⌘⌥E).

Clipboard & delete

  • Copy (⌘C) — Copies the selection to the internal clipboard.
  • Cut (⌘X) — Copies and removes the selection.
  • Paste (⌘V) — Pastes at playhead (or at selection / over selection / replace selection, depending on Edit → Paste Behaviour).
  • Delete (⌘⌫) — Removes the selection (no clipboard).
  • Duplicate (⌘D) — Inserts a copy of the selection immediately after the selection.

Processing

  • Silence (⌘⇧S) — Replaces selection with silence, or inserts silence before selection (Edit → Silence Operation).
  • Fade In / Fade Out (⌘⇧G / ⌘⇧H) — Apply fade; curve is chosen in Edit → Fade Curve (linear, logarithmic, exponential, S-curve).
  • Normalize (⌘⇧N) — Normalizes the selection to maximum level without clipping.
  • Invert phase (⌘⇧I) — Inverts the phase of the selection.
  • Reverse (⌘⇧R) — Reverses the selection in time.
  • Trim to selection — Keeps only the selection and discards the rest of the file (available from the edit toolbar).

Dual Edit Mode (varispeed + gain)

When enabled (experimental), ⌘⌥V or ⌘⌥↑/↓ enters Dual Edit Mode on the current selection. In this mode:

  • Keyboard: Left/Right arrows adjust varispeed (time stretch); Up/Down adjust gain. Return commits, Escape cancels.
  • Trackpad: ⌘⌥+drag on selection handles adjusts varispeed; Shift+drag adjusts gain. Changes commit on release. Snap to grid is optional (Edit → Dual Mode → Snap to Grid).

Dual mode maximum gain and resize (semitones) are set in Advanced Settings. Step mode (percentage, musical, or movement precision) is in Edit → Dual Mode and Edit → Movement Precision.

Undo / Redo

⌘Z Undo and ⌘⇧Z Redo apply to edit operations when Edit Operations are enabled.

Extract & export

Extract selection

Extract writes the current selection (or each segment between split points) to a new file using the format and options in Settings → Export. ⌘E extracts; ⌘⇧E extracts and opens the new file in the editor.

Extract options (in Export settings): normalize before extract, add short fades to avoid clicks, apply varispeed to export (if varispeed is enabled), copy source metadata (MP3/M4A only), and for unedited MP3 — lossless extraction (copy frames without re-encoding).

Export

“Export” typically refers to the same engine as Extract: the chosen time range (e.g. full file or selection) is written using the selected format, sample rate, bit depth/bitrate, and channel configuration. Export directory, filename suffix, and uniqueness (counter/timestamp) are configured in the Export section of Settings.

Advanced settings

Open Settings with ⌘,. Tabs typically include: General, Export, Appearance, and Advanced (or similarly named).

General

  • Last used open/export folders (and export directory bookmark).
  • Merge silence duration between merged files (Advanced → Merge Operations).

Export

As in “Specifications”: format, sample rate, bit depth, bitrate, channels, export directory, filename suffix, uniqueness method. Plus: normalize on extract, add fades on extract, apply varispeed to export, copy metadata (MP3/M4A), and lossless extraction for MP3 when applicable.

Appearance

  • Waveform color, selection color, selection opacity, playhead color, scrubbing color, split point color, background, overlay/button color.
  • Color scheme: System (time-based), Dark, or Light (inverted).

Advanced (experimental & behavior)

  • Edit operations — Enable/disable copy, cut, paste, and all edit processing. When disabled, Edit menu items are dimmed.
  • Scrubbing — Scrubbing is always available; here you may see scrubbing preview duration and trackpad/turntable options (or they’re in Playback menu).
  • Varispeed — Enable varispeed control; then set min/max rate (e.g. 0.1×–8×).
  • Dual Mode maximums — Max gain and max resize (semitones) for Dual Edit Mode.
  • Selection-centered zooming — Zoom centers on selection when enabled (View menu).
  • Selection centering during playback — When playing selection only, view can auto-center on the selection (View menu).
  • Movement precision — Category (Frames, Seconds, Percentage) is in Edit → Movement Precision; the exact step (sample, quarter frame, 10 ms, etc.) is in Playback → Movement Precision and via ⌥1–⌥9.
  • Paste behaviour, Fade curve, Silence operation — Set in Edit menu submenus.
