Parlante - Audio Waveform Editor

Edit audio with precision. Native macOS. Clean, fast, local.

What it does

A native macOS app for editing audio files. Load WAV, MP3, AIFF, M4A, FLAC, or extract audio from video. View a high-performance tile-based waveform, make selections, and apply edits. All processing happens on your Mac; no data is sent to external servers.

Features

  • Open & merge — Open single or multiple files; merge tracks into one.
  • Waveform view — Zoom and pan with a tile-based Metal-rendered waveform. Customize waveform, selection, and playhead colors.
  • Selection — Click-drag to select a range. Loop playback over selection. Clear with Escape or shortcut.
  • Playback — Play/pause, play selection only, jump to start/end. Scrub by dragging on the waveform (turntable-style).
  • Varispeed — Temporary pitch/speed change for playback (e.g. 0.5×–2×) without altering the file.
  • Edit operations — Copy, cut, paste, delete, duplicate, trim to selection, silence, fade in/out, normalize, reverse. Optional paste/silence behavior (replace vs insert) and fade curves.
  • Extract — Export the current selection as a new file. Command-click to extract and open in a new window.
  • Export — Save as WAV (8–32 bit) or MP3. Choose sample rate and bit depth where applicable.
  • Keyboard shortcuts — Full set of shortcuts for transport, selection, and edits (see in-app Keyboard Shortcuts).
  • Preferences — Last used folders, export defaults, waveform appearance, and other options.

Technical

Built with SwiftUI and AVFoundation. Waveform rendering uses Metal (GPU) and a tile-based cache for smooth zoom and pan. Supports light and dark mode and respects system appearance.
